# Tikslai

> Atnaujinti šį failą kiekvieno ketvirčio pradžioje.

## Q2 2026 (balandis–birželis)

*Formalių ketvirčio tikslų dar nėra — naudojame pagrindinius prioritetus kaip neformalius tikslus.*

### Darbas
- [ ] Sukurti pasiūlymų šabloną, kuris sutrumpintų procesą bent perpus
- [ ] Optimizuoti sąskaitų išrašymo procesą (perduoti buhalterams greičiau)
- [ ] Sukurti veikiančią darbo dienos struktūrą

### Asmeninis
- [ ] Nustatyti reguliarų fizinio aktyvumo grafiką
- [ ] Išgryninti bent 2–3 papildomo pajamų šaltinio idėjas

### Ateičiai (Q3+)
- Pradėti testuoti papildomą pajamų šaltinį
- Automatizuoti laiškų atrašymą šablonais
- Sukurti kainų skaičiavimo šabloną
